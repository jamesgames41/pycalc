# PyCalc: A Simple Python Calculator

PyCalc is a lightweight, easy-to-use CLI application designed for quick mathematical computations. Whether you need to solve basic arithmetic or calculate the area of 2D geometric shapes, PyCalc has you covered.

---

## üöÄ Features

* **Basic Arithmetic:** Perform addition, subtraction, multiplication, and division.
* **2D Geometry:** Quickly calculate the area for:
    * Squares
    * Rectangles
    * Circles
    * Triangles
* **User-Friendly Interface:** Simple text-based prompts to guide you through calculations.

## üìã Usage Examples

### Basic Math
When prompted, select the **Arithmetic** mode to solve equations like:
$25 \times 4 = 100$

### 2D Area Calculation
Select the **Geometry** mode. For a circle, the app uses the formula:
$$A = \pi r^2$$
*(Where $r$ is the radius provided by the user)*
